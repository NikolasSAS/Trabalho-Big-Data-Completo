# Análise de dados em Big Data da infraestrutura das escolas de Niterói ​

### Obgetivo do trabalho: O objetivo deste trabalho de extensão é apresentar uma análise da infraestrutura nas escolas de Niterói visando evidenciar alguns pontos. A quantidade de escolas que oferecem alimentação, quantidade de computadores e quantidade de laboratórios na última década (2015-2024).

### Dentro da pasta TRAB Big Data temos 3 arquivos 
- Pasta código
- Arquivo de Relatório
- Slides Big Data
### Pasta Código: Dentro da pasta temos o ProgramaV2.py, que é o código em si, e junto dele temos os arquivos Excel que o programa usa para realizar a análise de dados. 
- #### O programa escrito em Python lê os arquivos Excel e produz uma média em relação ao tópico escolhido (quantidade de computadores, quantidade de escolas que oferecem alimentação e quantidade de escolas com laboratório). Depois, o programa utiliza essas médias por ano para gerar uma Regressão Linear e uma Distribuição de Frequência.
### Arquivo de Relatório: Contém o relatório do projeto em formato pdf.
- #### O relatório descreve as etapas de Diagnóstico e Teorização, Planejamento e Desenvolvimento do projeto e Encerramento do Projeto.  
### Slides Big Data: Contém os slides em formato pdf.
- #### Contém os objetivos gerais do projeto, a base de dados utilizada, as tecnologias utilizadas e os resultados do código.
