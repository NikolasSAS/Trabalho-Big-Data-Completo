# Análise de dados em Big Data da infraestrutura das escolas de Niterói ​

### Obgetivo do trabalho: O objetivo deste trabalho de extensão é apresentar uma análise da infraestrutura nas escolas de Niterói visando evidenciar alguns pontos. A quantidade de escolas que oferecem alimentação, quantidade de computadores e quantidade de laboratórios na última década (2015-2024).

### Dentro da pasta TRAB Big Data temos 3 arquivos 
- Pasta código
- Arquivo de Relatório
- Slides Big Data
#### Pasta Código: Dentro da pasta temos o ProgramaV2.py, que é o código em si, e junto dele temos os arquivos Excel que o programa usa para realizar a análise de dados
#### Arquivo de Relatório: Contém o relatório completo em formato pdf
#### Slides Big Data: Contém os slides em formato pdf
